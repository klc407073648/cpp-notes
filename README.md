## 技术栈

- 开发框架：vue
- 组件库：vuepress
- 主语言：JavaScript
- 环境：腾讯云(nginx 部署)

## 快速上手

1. 下载项目:

```
git clone git@github.com:klc407073648/cpp-notes.git
```

2. 进入项目路径, 安装依赖: 

```
yarn install
```

3. 构建内容: 

```
vuepress build
```

4. 部署: 将docs下的文件copy至nginx服务器即可
