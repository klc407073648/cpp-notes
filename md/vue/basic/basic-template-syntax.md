# 模板语法

## 分类

1. 插值语法
    * 功能：用于解析**标签体**内容
    * 写法：`{{XXX}}`,`XXX`写的是**Js表达式**，且`XXX`可以自动读取到`data`中的所有属性

2. 指令语法

    * 功能：用于**解析标签**（包括：标签属性、标签体内容、绑定事件）
    * 举例 
      ```html
      <a v-bind:href="xxx"></a>
      <!--等价于-->
      <a :href="xxx"></a>
      <!--
          xxx写js表达式，且可以读取data中的所有属性
          Vue将xxx当成js表达式执行
          -->
      ```
